# ГРИБОК vs МАРИО

## 🍄 О чём это вообще

Сокращённо — **РЬОУ!**

Когда надоело спасать принцесс в чужих замках, а душа просит грибного правосудия. Это шуточная пиксельная аркада-перевёртыш, где **ты — гриб**. Не какой-нибудь там говорящий гриб с проблемами самоидентификации, а — **внимание** — квадратный гриб в кепке, который решил, что **лучше сдохнуть ебучим ноунеймом, чем прославиться и стать Марио**.

Высокомерный птеродактиль (Оксимирон) где-то на фоне. Сделайте вид, что это важно для сюжета.

Твоя цель — прыгать сверху на Марио. Потому что, блядь, **со смыслом нахуй, это славянская словесность**.

## 🎮 Управление

- `← →` или `A D` — бегать. Твоя мать пропускает всех как дверной косяк, а ты попробуй хотя бы грибом по экрану по двигай.
- `Пробел` / `↑` / `W` — прыгать. Интегральная гипербола ректальной кибернетики, не иначе.
- Тач-кнопки на мобилке — для тех, кто хочет быть грибом в автобусе.

## 🎯 Цель

Убить **10 Марио** прыжками сверху. **Ты выполнила задание на пять с плюсом!** — сказал бы тебе голос в голове, если бы он у тебя был. Сбоку Марио не трогать — иначе потеря жизни. **Клоун задумался о своём наследии? Так вот оно — на твоей кровати ебутся педики.** А у тебя — 3 жизни.

Жизней, Карл. Три.

## 🛠 Как запустить

У меня тоже есть птички, и у этих птичек — микрофоны-петлички. Но тебе они не нужны. Просто открой `index.html` браузером. Никаких `npm install`, никакой боли.

**Помнишь, ты съел деда? Сегодня дед съел Моргенштерна.** А ты просто открываешь файл.

## 🌐 Хостинг на GitHub Pages

1. Заводишь репозиторий на GitHub.
2. Кидаешь туда `index.html` — через веб-интерфейс, гитом, да хоть телепатией.
3. **Settings → Pages.**
4. В **Source** выбираешь ветку `main` (или умри).
5. Жмёшь **Save**. Идешь пить компот. Через минуту игра доступна по адресу: `https://<твой-username>.github.io/<имя-репо>/`

Игра целиком в одном файле (HTML + CSS + JS + canvas-графика). Никаких внешних ресурсов. GitHub Pages всё стерпит.
